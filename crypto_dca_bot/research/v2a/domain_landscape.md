## Part 1:核心共識(所有學派都認同)

1. **市場大部分隨機,edge 又小又稀缺** — 沒人能預測明天漲跌。能做的是找微小統計偏向,靠重複累積。
2. **風險管理 > 預測準度** — 活下來比賺錢重要。「押多大」+「最大虧損控制」比訊號準度更決定長期結果。
3. **過度擬合(死背歷史)是頭號敵人** — 歷史表現好 ≈ 沒意義,要看沒看過的資料上的表現。
4. **交易成本真實、會殺死策略** — 手續費 + 滑點 + 衝擊成本,紙上績效扣成本後常由正轉負。
5. **看「過程」不看「單次結果」** — 結果有運氣成分。系統化交易的全部意義是把情緒移除。
6. **沒有聖杯,所有 edge 會衰退** — 賺錢方法被夠多人發現就失效,必須持續研究。

**濃縮一句:** 量化交易是「機率遊戲 + 生存遊戲」,不是「預測遊戲」。找微小統計優勢、用嚴格風控活夠久、讓大數法則替你工作,同時時刻警惕自欺。

## Part 2:三大爭議

### 爭議 1:市場能不能被打敗?

| | 效率市場派 | 可打敗派 |
|---|---|---|
| 主張 | 價格已反映所有資訊,長期沒人穩定贏大盤 | 市場有結構性低效率,可捕捉 |
| 最硬論據 | 80%+ 主動型基金長期輸大盤;贏家多半是運氣 + 倖存者偏差 | 文藝復興 Medallion 基金連 30 年年化 ~66%;動能 / 價值等 factor 學術界驗證幾十年 |

**對 V2:** 做 V2 = 押注「有可捕捉的低效率」。但連可打敗派都承認 edge 小、稀缺、會衰退。

### 爭議 2:策略需不需要「說得通的理由」?

| | 可解釋派 | 純統計派 |
|---|---|---|
| 主張 | 策略必須有經濟學上說得通的理由 | 統計顯著就用,理由其次 |
| 最硬論據 | 沒理由的 pattern 就是噪音;要求 rationale 是防過度擬合最強防線 | 人對「為什麼」常事後編故事;堅持要故事會錯過反直覺的真 edge |

**對 V2:** 起步三策略(趨勢 / 均值回歸 / 宏觀)都是可解釋派友善 — 都有經濟學故事,保守穩健。

### 爭議 3:簡單還是複雜?(直接打在 V2 設計核心)

| | 簡單派 | 分散派 |
|---|---|---|
| 主張 | 少數幾個穩健策略 | 大量低相關策略疊加 |
| 最硬論據 | 參數越少越不易過度擬合;2008 / 2020 崩盤精密模型死最慘(LTCM 爆倉) | 單一策略再穩健都有失效時;「分散化是唯一免費午餐」;文藝復興 / Two Sigma 跑上千訊號 |

**對 V2:** 見下方 Part 3 使用者定調。

## Part 3:使用者立場(2026-05-12 拍板)

- **爭議 1:無意見** — 主動交易者本就已選邊(押注市場可打敗),不糾結此題。
- **爭議 2:無意見** — 「策略很多是自證預言(夠多人相信並照做就成真),這本身是一類 edge 來源」,故「要不要經濟學理由」對使用者無實益。
- **爭議 3:定調簡單派**(使用者委託 Claude 專業判斷):
  - V2 實際策略數目標 anchor 在 **3 個**;roadmap「3-7」中 7 為理論上限、不當目標。
  - 「簡單」不只指數量,每個策略內部也簡單:參數理想 < 5 個、邏輯一句話講得清。
  - 理由:複雜派玩法需規模才成立(文藝復興級資源);個人玩家頭號死因是複雜度爆炸,非分散不足。但「只有 1 個策略」也危險(命運綁單一市場氣候),故保留最低限度分散。
  - 一句話:**複雜度要配得上你的資源。**
